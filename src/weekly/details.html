Week 1: Introduction to HTML

Starts on: 2024-01-08

Description & Notes

This week covers the fundamental building blocks of the web: HTML. We will explore semantic tags, document structure, and basic elements like headings, paragraphs, links, and images.

Key learning objectives:

  • Understand the basic structure of an HTML document
  • Learn to use semantic HTML5 elements
  • Create links, images, lists, and tables
  • Practice with forms and input elements
  • Learn about accessibility best practices

Note: Complete the practice exercises before the next class. The assignment will be posted on the course portal by Wednesday.
